== Description ==
This plugin gives you the functionality to get information about the blog used in the Pressengine application.


= Features:  =

* Get number of posts in your blog
* Get number of pages in your blog
* Get number links in your blog
* Get number of words in your blog
* Get date of last submitted post in your blog,
* Get number of posts in your blog\'s frontpage
* Get date of last submitted post in your blog\'s frontpage,

This plugin is mainly used for the blogs which is added to the The Pressengine Application.

== Installation ==

1. Download the plugin,
2. Unzip to /wp-contents/plugin
3. Goto to yourdomain.com/wp-admin/, click plugins, activate the plugin.
